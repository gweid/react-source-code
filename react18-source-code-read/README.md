# react 源码阅读



## 启动调试



进入项目 `react18-source-code-read` 根目录

```shell
cd react18-source-code-read
```



安装包：

```shell
yarn
```



启动：

```shell
yarn start
```



## 目录结构

```
react18-source-code-read/
  ├── config                            // 工程化配置
  ├── scripts                           // 工程化配置
  ├── src                               // 子包目录
  │   ├── react                         // react 源码
  │   ├── components                    // 调试代码（组件）
  │   ├── pages                         // 调试代码（页面）
  │   ├── store                         // 调试代码（redux）
  │   └── index.js                      // 调试代码（入口文件）
```
